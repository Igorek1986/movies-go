Политика обработки персональных данных

Настоящая Политика определяет порядок обработки персональных данных пользователей сервиса {{with .Data}}{{.SiteName}}{{end}} (далее — Сервис).

1. Какие данные мы обрабатываем

  • Имя пользователя (логин)
  • Telegram ID и username — при добровольной привязке аккаунта Telegram
  • IP-адрес и User-Agent браузера — для защиты аккаунта и ведения сессий

2. Цели обработки

  • Аутентификация и управление сессиями
  • Восстановление пароля через Telegram
  • Отправка уведомлений о входе в аккаунт и истечении подписки

3. Хранение данных

Данные хранятся на сервере оператора. Telegram ID и username хранятся только при наличии явно данного согласия и удаляются при отвязке Telegram-аккаунта.

4. Передача третьим лицам

Данные не передаются третьим лицам, за исключением технически необходимого взаимодействия с платформой Telegram (при привязке аккаунта).

5. Права субъекта данных

Вы вправе в любой момент отвязать Telegram-аккаунт и удалить свою учётную запись в разделе «Настройки аккаунта».

6. Использование cookie-файлов

Сайт использует cookie-файлы для обеспечения работы сессий авторизации и хранения пользовательских предпочтений (например, тема оформления). Cookie не передаются третьим лицам и не используются для отслеживания поведения пользователей без их согласия.

Сторонние аналитические инструменты на сайте отключены.
